Perguntas Frequentes (FAQ)

O botão "Login" no Navbar está disponível exclusivamente para funcionários. Se você é um funcionário registrado, clique em "Login" e insira suas credenciais para acessar o Dashboard. Caso ainda não tenha uma conta de funcionário, entre em contato com a administração.

Para encontrar nosso catálogo, navegue até a seção "Catálogo" no menu principal. Lá, você encontrará uma variedade de produtos organizados por categorias.

Para explorar nosso catálogo, vá até a seção "Catálogo" no menu principal. Lá, você encontrará uma variedade de produtos organizados por categorias. Use a barra de pesquisa para encontrar itens específicos ou utilize os filtros "Tipo", "Tecido", "Marca", "Gênero" e "Tamanho" para refinar sua busca.

No momento, apenas funcionários têm a opção de fazer login. Os usuários ainda não podem se registrar, mas essa funcionalidade pode ser implementada no futuro. Fique atento às atualizações.

O Dashboard é uma área exclusiva para funcionários, onde é possível gerenciar produtos e outras informações. Ao fazer login como funcionário, você terá acesso a ferramentas para adicionar, editar ou remover produtos do catálogo, entre outras funcionalidades administrativas.

Para entrar em contato com a Polaris, visite a seção "Polaris" no footer. Lá, você encontrará um link que o levará ao site oficial da Polaris, onde poderá obter informações de contato.

Os links para os "Termos de Uso" e "Políticas de Privacidade" estão disponíveis no footer do site. Ao clicar nesses links, você terá acesso a informações detalhadas sobre as políticas e termos aplicáveis ao uso do site.

Se precisar de mais ajuda ou tiver dúvidas não abordadas aqui, sinta-se à vontade para entrar em contato conosco. Utilize o link "Ajuda" no footer para acessar a seção de FAQ/Ajuda ou entrar em contato para informações adicionais.
